# Kişisel Web Sitem

Bu repository, benim kişisel web sitemin kaynak kodunu içermektedir.

Site; kendimi tanıttığım, yaptığım projeleri paylaştığım, oyun aktivitelerimi gösterdiğim ve internet üzerindeki varlığımı tek bir yerde topladığım kişisel bir merkez olarak tasarlanmıştır.

## 🌐 Hakkında

Bu web sitesi ziyaretçilerin beni daha yakından tanıyabilmesi, geliştirdiğim projeleri inceleyebilmesi ve farklı platformlardaki hesaplarıma kolayca ulaşabilmesi amacıyla oluşturulmuştur.

Site içerisinde aşağıdaki bölümler bulunmaktadır:

* Kişisel tanıtım
* Yazılım bilgileri ve kullandığım teknolojiler
* Geliştirdiğim projeler
* Steam oyun istatistiklerim
* Kişisel notlar
* Sosyal medya hesaplarım

## ⚡ Özellikler

### 👤 Kişisel Tanıtım

Kim olduğum, ilgilendiğim alanlar ve neler yaptığım hakkında bilgiler içerir.

### 💻 Yazılım Bilgileri

Bildigim veya kullandığım programlama dilleri ve teknolojiler listelenir.

### 🚀 Projeler

Geliştirdiğim veya katkıda bulunduğum projelerin yer aldığı bölüm.

### 🎮 Steam Entegrasyonu

Steam hesabımdaki bazı bilgiler sitede gösterilir:

* Oynadığım oyunlar
* Oyun saatleri
* Oyun başarımları

### 📝 Notlar

Zaman zaman paylaştığım küçük notlar, fikirler veya ilgimi çeken şeyler.

### 🔗 Sosyal Medya

Sosyal medya hesaplarıma hızlı erişim sağlayan bağlantılar.

## 🛠 Kullanılan Teknolojiler

Site modern web teknolojileri kullanılarak geliştirilmiştir.

Örneğin:

* HTML
* CSS
* JavaScript
* API entegrasyonları

## 📄 Lisans

Bu projedeki kaynak kodlar **MIT Lisansı** altında paylaşılmaktadır.

Ancak web sitesinde bulunan kişisel içerikler, tasarım öğeleri ve marka unsurları izinsiz olarak kopyalanamaz veya başka bir yerde kullanılamaz.
